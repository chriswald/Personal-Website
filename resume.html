UWPClasses

A program rewritten for Professor Robert W. Hasker to be cross-platform. UWPClasses is used by students and staff at the University of Wisconsin - Platteville to create class schedules and view information about classes. The project consisted of two portions. First was rewriting the entire frontend with a focus on making it cross-platform. Second was rewriting much of the internal code with an eye for future maintainability.

Western Time

Western Time is a project closely related to my work on UWPClasses. It is a web based version of the program that aims to provide all of the functionality of UWPClasses, but in a more convenient way. Western Time was developed on my own time and not in conjunction with UWPClasses.

F.aceBash

Uses the Facebook API and libcurl to interact with Facebook in a command line environment. The applications when used alone are limited, however when combined with other technologies the possibilites grow. Already it has been used in combination with a system that reads text (SMS) messages to allow feature phones to interact with Facebook.
